Projeto Bloco de Notas

Projeto de exemplo sobre framework Angular. Características do framework que são abordadas:

  • Criação e execução de projetos
  • Criação de classes de dados
  • Criação de componentes
  • Recurso data-binding
  • Diretivas de template (ngFor)
  • Uso de pipes
  • Recurso mestre-detalhe entre componentes (@Input)
  • Navegação interna entre páginas
  • Serviços de interface (providers)
  • Acesso a recursos externos
